Большинство архивных химических книг и сканированных материалов характеризуются неоднородным качеством. В работе~\cite{refAkhil} выделялись следующие проблемы качества сканированных книг:
\begin{itemize}
	\item Низкое разрешение / размытость (рис.~\ref{fig:poor_quality} а)
	\item Шум, зернистость, артефакты сжатия (рис.~\ref{fig:poor_quality} б)
	\item Неравномерная засветлённость / тени (рис.~\ref{fig:poor_quality} в)
	\item Ориентация страницы с малым поворотом (рис.~\ref{fig:poor_quality} г)
\end{itemize}
